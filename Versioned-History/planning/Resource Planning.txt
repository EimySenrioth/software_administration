Resource Planning
Resource planning for the "AcBi" project allocates human, material, 
and financial assets to achieve the development 
objectives for the mobile/PC application within a 6-month timeframe.

Technologies
The following technologies were selected for the development of the "AcBi" project, as shown in the Table 1 & 2:

Table 1. Technologies
| Category                       | Technology                       | Purpose/Goal                                                                                                                             |
| ------------------------------ | -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Frontend (UI & Visualizations) | React (with JavaScript)          | Build the main dashboard, timelines, and interactive charts.                                                                             |
| Frontend (UI & Visualizations) | vis.js & D3.js                   | Create dynamic diagrams.                                                                                                                 |
| Backend/Core (Business Logic)  | Rust                             | Implement the obfuscated core, including entities (e.g., Officials, Companies) and main modules; protection against reverse engineering. |
| Backend/Core (Business Logic)  | Serde & Reqwest                  | Handle JSON serialization and resilient HTTP clients for the API Config Manager.                                                         |
| Backend/Core (Business Logic)  | Go & Python                      | Simpler concurrent logic for modules such as Process Tracking.                                                                           |
| Backend/Core (Business Logic)  | C++                              | Used in high-performance critical sections.                                                                                              |
| Framework & Multiplatform      | Tauri 2.0 Mobile                 | Bridge between frontend (React) and backend (Rust), handling IPC/Commands.                                                               |
| Framework & Multiplatform      | Ionic                            | Web-native integration for interactive UI.                                                                                               |
| Framework & Multiplatform      | React Native (Option 2)          | Mobile-first approach without Tauri, ideal for native UI in performance-heavy environments.                                              |
| Database & Storage             | SQLCipher/rusqlite (AES-256-GCM) | Encrypted local storage.                                                                                                                 |
| Database & Storage             | Realm                            | Embedded mobile DB for fast queries in high-volume local data scenarios.                                                                 |
| Security & Cryptography        | SHA Family                       | Hash generation.                                                                                                                         |
| Security & Cryptography        | wolfSSL                          | VPN support.                                                                                                                             |
| Security & Cryptography        | Custom Rust (Anti-forensics)     | Sandbox and memory dump detection.                                                                                                       |
| Security & Cryptography        | Obfuscator.io (JS)               | Code obfuscation for frontend/backend to prevent reverse engineering.                                                                    |
| Infrastructure & Resilience    | Local-first                      | Full offline operation; resilient API client via Tor for external public registries.                                                     |
| Testing & Quality              | Jest (JS) & Cargo test (Rust)    | Unit testing for frontend and backend code.                                                                                              |
| Testing & Quality              | Appium                           | E2E testing for mobile environments (e.g., API Manager flows).                                                                           |
| Testing & Quality              | OWASP ZAP                        | Security testing for mobile vulnerabilities and E2EE validation.                                                                         |
| Testing & Quality              | Cypress / Burp Suite / SonarQube | Alternatives for integration testing, ethical hacking, and static code analysis.                                                         |
